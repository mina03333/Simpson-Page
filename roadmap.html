Start with JAVA

kh정보교육원에서 첫 프로그래밍 언어 자바를 배우다!

Database with Oracle

관계형 데이터베이스로 오라클을 배우고, JAVA API인 JDBC를 이용해 자바와 DB연동을 배우다. 후에 MYSQL과 NoSQL Database인 MongoDB도 써볼 수 있는 기회가 있으면 좋겠다.

FRONT-END

HTML, CSS, Javascript, JQuery, Web Storage를 배우다. 이번에 홈페이지를 만들기 위해 개인적으로 CSS전처리기인 SCSS를 사용해보았다. 그리고 Node.js와 Package Manager인 NPM을 설치하고, Build Tool로 gulp를 사용해보았다.

앞으로는 자바스크립트를 제대로 배워보고 싶다. 그래서 Framework인 React.js도 써보고 싶고, Package Manager, Bundler에 대한 개념도 제대로 잡고 싶다. 기회가 된다면 자바스크립트 서버를 다루는 Node.js도 배워서 Java와 비교해보아도 좋을 것 같다.

Basic & DevOps

Git, Github, 자료구조, 알고리즘, 디자인 패턴, Linux, Terminal, Networking 등 개발자로써 기본적이고 필수적으로 알아야하는 부분들도 차근차근 공부할 것이다.

In The Future

어디에서 무엇을 만들든 기본기가 탄탄하고, FRONT와 BACKEND를 아우를 수 있는 개발자가 되고 싶다. 하루 아침에 그런 멋진 개발자가 될 수는 없겠지만, 인내심을 갖고 시간과 노력을 들이면 언젠가는 내가 바라는 개발자가 될 수 있으리라 믿는다!
